# Kişiselleştirilmiş Haber Uygulaması

- 🔐 Kullanıcı kaydı ve girişi
- 📰 Kişiselleştirilmiş haber akışı
- 🎯 Kategori bazlı haber filtreleme
- 🌐 Türkçe haber içeriği
- 📱 Responsive tasarım

## Teknolojiler

### Frontend
- React.js
- React Router
- Axios
- Context API
- CSS3

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- NewsAPI.org
